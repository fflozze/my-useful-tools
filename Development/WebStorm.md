# WebStorm (JetBrains)

> [!TIP]
> [Passer à la version française](#version-française)

## Description
WebStorm is a powerful Integrated Development Environment (IDE) dedicated to JavaScript development and its ecosystem. Developed by JetBrains, it offers intelligent coding assistance for JavaScript, TypeScript, React, Vue, Angular, Node.js, HTML, and stylesheets. It is renowned for its ability to understand code structure, facilitating refactoring and debugging.

## Website / Link
- [Official Website](https://www.jetbrains.com/webstorm/)
- [Documentation](https://www.jetbrains.com/help/webstorm/)

## Installation
Via JetBrains Toolbox (Recommended) or direct download:
```bash
# Download and install JetBrains Toolbox App to manage JetBrains tools
https://www.jetbrains.com/toolbox-app/
```

## Useful Commands / Tips
- `Double Shift`: Search Everywhere.
- `Alt + Enter`: Show Context Actions and Quick Fixes.
- `Ctrl + Click` (or `Ctrl + B`): Go to Declaration.
- `Ctrl + Alt + L`: Reformat Code.
- `Shift + F6`: Rename Symbol (Refactoring).
- `Ctrl + Shift + F`: Find in Files.

## Configuration
- **Settings/Preferences**: `Ctrl + Alt + S` - To configure linters (ESLint, Prettier), themes, and plugins.
- **Plugins**: WebStorm has a rich ecosystem of plugins to extend its functionality (Marketplace in settings).

## Notes
- WebStorm is a paid product (subscription), but free for students and open source projects under certain conditions.
- It includes many tools for web development out of the box without requiring heavy configuration of third-party plugins (unlike VS Code).

# Version Française

## Description
WebStorm est un environnement de développement intégré (IDE) puissant dédié au développement JavaScript et à son écosystème. Développé par JetBrains, il offre une assistance au codage intelligente pour JavaScript, TypeScript, React, Vue, Angular, Node.js, HTML et stylesheets. Il est réputé pour sa capacité à comprendre la structure du code, facilitant ainsi la refactorisation et le débogage.

## Site Web / Lien
- [Site Officiel](https://www.jetbrains.com/webstorm/)
- [Documentation](https://www.jetbrains.com/help/webstorm/)

## Installation
Via JetBrains Toolbox (Recommandé) ou téléchargement direct :
```bash
# Télécharger et installer JetBrains Toolbox App pour gérer les outils JetBrains
https://www.jetbrains.com/toolbox-app/
```

## Commandes Utiles / Astuces
- `Double Shift` : Rechercher partout (Search Everywhere).
- `Alt + Enter` : Afficher les actions contextuelles et les corrections rapides.
- `Ctrl + Click` (ou `Ctrl + B`) : Aller à la déclaration.
- `Ctrl + Alt + L` : Formater le code.
- `Shift + F6` : Renommer un symbole (Refactoring).
- `Ctrl + Shift + F` : Rechercher dans tout le projet.

## Configuration
- **Settings/Preferences** : `Ctrl + Alt + S` - Pour configurer les linters (ESLint, Prettier), les thèmes, et les plugins.
- **Plugins** : WebStorm possède un riche écosystème de plugins pour étendre ses fonctionnalités (Marketplace dans les paramètres).

## Notes
- WebStorm est un produit payant (abonnement), mais gratuit pour les étudiants et les projets open source sous certaines conditions.
- Il inclut de base de nombreux outils pour le développement web sans nécessiter une configuration lourde de plugins tiers (contrairement à VS Code).

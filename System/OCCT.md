# OCCT (OverClock Checking Tool)

> [!TIP]
> [Passer à la version française](#version-française)

## Description
OCCT is the most popular all-in-one stability check and stress test tool available. It is designed to generate heavy loads on your components (CPU, GPU, Memory, Power Supply) to detect hardware errors, overclocking instability, or cooling issues. It is widely used by overclockers and system builders to ensure stability.

## Website / Link
- [Official Website](https://www.ocbase.com/)
- [Download](https://www.ocbase.com/download)

## Installation
OCCT is portable and does not require a full installation.
1. Download the executable from the official website.
2. Run the `.exe` file directly.

## Useful Commands / Tips
- **CPU Test**: Use the "Large Data Set" mode for maximum error detection (memory controller/RAM focus) or "Small Data Set" for maximum heat generation (CPU core focus).
- **Power Test**: This test loads both the CPU and GPU simultaneously. It is excellent for testing your Power Supply Unit (PSU) and overall system cooling capabilities.
- **Monitoring**: OCCT includes a powerful monitoring engine (similar to HWMonitor) to view temperatures, voltages, and fan speeds during tests.

## Configuration
- **Safety Limits**: You can configure temperature thresholds (e.g., stop test if CPU > 90°C) to prevent hardware damage during stress testing.
- **Test Duration**: For a quick check, 10-15 minutes is usually enough. For rock-solid stability validation, run tests for 1 hour or more.

## Notes
- The "Personal" version is free for non-commercial use.
- Always monitor your temperatures closely when running stress tests for the first time on a new system.

# Version Française

## Description
OCCT est l'outil de vérification de stabilité et de test de charge (stress test) tout-en-un le plus populaire. Il est conçu pour générer de lourdes charges sur vos composants (CPU, GPU, Mémoire, Alimentation) afin de détecter les erreurs matérielles, l'instabilité de l'overclocking ou les problèmes de refroidissement. Il est largement utilisé par les overclockers et les monteurs de PC pour assurer la stabilité.

## Site Web / Lien
- [Site Officiel](https://www.ocbase.com/)
- [Téléchargement](https://www.ocbase.com/download)

## Installation
OCCT est portable et ne nécessite pas d'installation complète.
1. Téléchargez l'exécutable depuis le site officiel.
2. Lancez le fichier `.exe` directement.

## Commandes Utiles / Astuces
- **Test CPU** : Utilisez le mode "Large Data Set" pour une détection maximale des erreurs (focus contrôleur mémoire/RAM) ou "Small Data Set" pour une génération maximale de chaleur (focus cœurs CPU).
- **Test Power** : Ce test charge simultanément le CPU et le GPU. Il est excellent pour tester votre bloc d'alimentation (PSU) et les capacités globales de refroidissement du système.
- **Monitoring** : OCCT inclut un puissant moteur de surveillance (similaire à HWMonitor) pour voir les températures, les tensions et les vitesses des ventilateurs pendant les tests.

## Configuration
- **Limites de sécurité** : Vous pouvez configurer des seuils de température (par exemple, arrêter le test si CPU > 90°C) pour éviter d'endommager le matériel pendant les tests de stress.
- **Durée du test** : Pour une vérification rapide, 10-15 minutes suffisent généralement. Pour une validation de stabilité absolue, lancez les tests pendant 1 heure ou plus.

## Notes
- La version "Personnelle" est gratuite pour un usage non commercial.
- Surveillez toujours vos températures de près lorsque vous lancez des stress tests pour la première fois sur un nouveau système.

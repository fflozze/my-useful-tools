# Brave Browser

> [!TIP]
> [Passer à la version française](#version-française)

## Description
Brave is a free and open-source web browser developed by Brave Software, Inc. based on the Chromium web browser. It is a privacy-focused browser that automatically blocks online advertisements and website trackers in its default settings. It also provides users the option to turn on Brave Rewards to earn Basic Attention Tokens (BAT) by viewing privacy-respecting ads.

## Website / Link
- [Official Website](https://brave.com/)
- [GitHub Repository](https://github.com/brave/brave-browser)

## Installation
```bash
# Windows (via Winget)
winget install Brave.Brave

# macOS (via Homebrew)
brew install --cask brave-browser

# Linux (Debian/Ubuntu)
sudo apt install brave-browser
```
Or download the installer from the [official website](https://brave.com/download/).

## Useful Commands / Tips
- **Brave Shields**: Click the lion icon in the address bar to toggle protections for a specific site.
- **Private Window with Tor**: Open a new private window with Tor (Alt+Shift+N on Windows) to hide your IP address from the sites you visit.
- **Sync**: Go to `Settings > Sync` to link your bookmarks and passwords across devices.
- **Wallet**: Access the built-in crypto wallet via the wallet icon or `brave://wallet`.

## Configuration
- **Settings**: Access global settings at `brave://settings/`.
- **Flags**: Enable experimental features at `brave://flags/`.
- **Extensions**: Compatible with Chrome Web Store extensions.

## Notes
- Brave is significantly faster than Chrome due to the blocking of trackers and ads.
- It is built on Chromium, so it supports almost all Chrome extensions.

# Version Française

## Description
Brave est un navigateur web libre et open-source développé par Brave Software, Inc. basé sur le navigateur web Chromium. C'est un navigateur axé sur la confidentialité qui bloque automatiquement les publicités en ligne et les traqueurs de sites web par défaut. Il offre également aux utilisateurs la possibilité d'activer Brave Rewards pour gagner des Basic Attention Tokens (BAT) en visionnant des publicités respectueuses de la vie privée.

## Site Web / Lien
- [Site Officiel](https://brave.com/fr/)
- [Dépôt GitHub](https://github.com/brave/brave-browser)

## Installation
```bash
# Windows (via Winget)
winget install Brave.Brave

# macOS (via Homebrew)
brew install --cask brave-browser

# Linux (Debian/Ubuntu)
sudo apt install brave-browser
```
Ou téléchargez l'installateur depuis le [site officiel](https://brave.com/fr/download/).

## Commandes Utiles / Astuces
- **Brave Shields** : Cliquez sur l'icône de lion dans la barre d'adresse pour activer/désactiver les protections pour un site spécifique.
- **Fenêtre Privée avec Tor** : Ouvrez une nouvelle fenêtre privée avec Tor (Alt+Maj+N sur Windows) pour masquer votre adresse IP aux sites que vous visitez.
- **Sync** : Allez dans `Paramètres > Sync` pour lier vos marque-pages et mots de passe entre vos appareils.
- **Portefeuille** : Accédez au portefeuille crypto intégré via l'icône de portefeuille ou `brave://wallet`.

## Configuration
- **Paramètres** : Accédez aux paramètres globaux via `brave://settings/`.
- **Flags** : Activez des fonctionnalités expérimentales via `brave://flags/`.
- **Extensions** : Compatible avec les extensions du Chrome Web Store.

## Notes
- Brave est nettement plus rapide que Chrome grâce au blocage des traqueurs et des publicités.
- Il est construit sur Chromium, il supporte donc presque toutes les extensions Chrome.

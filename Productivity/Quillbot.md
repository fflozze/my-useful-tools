# Quillbot

> [!TIP]
> [Passer à la version française](#version-française)

## Description
Quillbot is an AI-powered paraphrasing and grammar checking tool. It helps you rewrite text to improve clarity, vocabulary, and tone. It is widely used by students, writers, and professionals to enhance their writing and avoid plagiarism by rephrasing content.

## Website / Link
- [Official Website](https://quillbot.com)
- [Chrome Extension](https://chrome.google.com/webstore/detail/quillbot-for-chrome/iidnbdjijdkbmajdffnidomddglmieko)

## Installation
- **Web**: Accessible directly via the browser.
- **Extensions**: Available for Chrome, Word, and macOS.
```bash
# Install the browser extension for seamless integration.
```

## Useful Commands / Tips
- **Paraphraser**: Rewrite sentences or paragraphs in different styles (Standard, Fluency, Formal, Simple, Creative, Expand, Shorten).
- **Grammar Checker**: Fix grammar, spelling, and punctuation errors.
- **Plagiarism Checker**: Scan your text to ensure originality (Premium).
- **Summarizer**: Condense long articles or documents into key points.
- **Synonym Slider**: Adjust the vocabulary change frequency to find the perfect word choice.

## Configuration
- **Dialect**: Choose between US, UK, Canadian, and Australian English.
- **Extensions**: Integrate directly into Google Docs or Microsoft Word for real-time suggestions.

## Notes
- The free version is powerful but has character limits and fewer modes. Premium unlocks all modes and higher limits.
- Excellent for non-native speakers to polish their English writing.

# Version Française

## Description
Quillbot est un outil de reformulation et de correction grammaticale utilisant l'IA. Il vous aide à réécrire du texte pour améliorer la clarté, le vocabulaire et le ton. Il est largement utilisé par les étudiants, les rédacteurs et les professionnels pour améliorer leurs écrits et éviter le plagiat en reformulant le contenu.

## Site Web / Lien
- [Site Officiel](https://quillbot.com)
- [Extension Chrome](https://chrome.google.com/webstore/detail/quillbot-for-chrome/iidnbdjijdkbmajdffnidomddglmieko)

## Installation
- **Web** : Accessible directement via le navigateur.
- **Extensions** : Disponible pour Chrome, Word et macOS.
```bash
# Installez l'extension de navigateur pour une intégration fluide.
```

## Commandes Utiles / Astuces
- **Paraphraseur** : Réécrivez des phrases ou des paragraphes dans différents styles (Standard, Fluide, Formel, Simple, Créatif, Développer, Raccourcir).
- **Correcteur Grammatical** : Corrigez les erreurs de grammaire, d'orthographe et de ponctuation.
- **Détecteur de Plagiat** : Analysez votre texte pour garantir son originalité (Premium).
- **Résumeur** : Condensez de longs articles ou documents en points clés.
- **Curseur de Synonymes** : Ajustez la fréquence de changement de vocabulaire pour trouver le mot parfait.

## Configuration
- **Dialecte** : Choisissez entre l'anglais américain, britannique, canadien et australien (et d'autres langues supportées).
- **Extensions** : Intégrez-le directement dans Google Docs ou Microsoft Word pour des suggestions en temps réel.

## Notes
- La version gratuite est puissante mais a des limites de caractères et moins de modes. La version Premium débloque tous les modes et des limites plus élevées.
- Excellent pour les non-natifs pour peaufiner leur écriture en anglais (et autres langues).

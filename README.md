# Useful Tools

Ce dépôt a pour but de présenter les outils que j'utilise au quotidien. Chaque outil possède sa propre fiche détaillée (fichier `.md`) regroupant une description, des liens utiles, l'installation et des astuces.

## Structure

Les outils sont triés par catégorie dans des dossiers :

- **Development/** : Outils de développement, IDE, frameworks, etc.
- **Productivity/** : Outils de gestion de temps, notes, bureautique.
- **System/** : Outils système, terminaux, monitoring.
- **Design/** : Outils de conception graphique, UI/UX.
- **Utilities/** : Autres utilitaires pratiques.

## Contribuer / Ajouter un outil

Pour ajouter un nouvel outil :
1. Copiez le fichier `_template.md`.
2. Renommez-le avec le nom de l'outil.
3. Placez-le dans le dossier correspondant à sa catégorie.
4. Remplissez les informations.

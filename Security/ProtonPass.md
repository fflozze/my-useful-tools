# Proton Pass

> [!TIP]
> [Passer à la version française](#version-française)

## Description
Proton Pass is an end-to-end encrypted password manager designed by the team behind Proton Mail. It secures your passwords, login credentials, and email aliases (Hide-my-email) to protect your online identity. It offers a user-friendly interface and seamless integration with the Proton ecosystem.

## Website / Link
- [Official Website](https://proton.me/pass)
- [GitHub (Applications)](https://github.com/protonprivacy)

## Installation
Available as a browser extension (Chrome, Firefox, Brave, Edge) and mobile application (iOS, Android).
```bash
# Install via your browser's extension store or mobile app store.
```

## Useful Commands / Tips
- **Hide-my-email**: Create unique email aliases for each service to reduce spam and protect your real email address.
- **Secure Notes**: Store sensitive information (PIN codes, recovery keys, credit card details) in encrypted notes.
- **Integrated 2FA**: Generate two-factor authentication codes directly within the app for faster logins.
- **Vaults**: Organize your logins and notes into different vaults (e.g., Personal, Work) for better management.

## Configuration
- **Synchronization**: Automatically syncs across all devices connected to the Proton account.
- **Import**: Easily import passwords from other browsers (Chrome, Firefox) or password managers (LastPass, 1Password, Bitwarden).
- **Auto-lock**: Configure the app to lock automatically after a period of inactivity.

## Notes
- Perfect for those already in the Proton ecosystem.
- The "Hide-my-email" feature is particularly useful for signing up for newsletters or temporary services without exposing your main email.
- It is open-source and audited, ensuring transparency and security.

# Version Française

## Description
Proton Pass est un gestionnaire de mots de passe chiffré de bout en bout, conçu par l'équipe derrière Proton Mail. Il sécurise vos mots de passe, vos identifiants de connexion et vos alias d'email (Hide-my-email) pour protéger votre identité en ligne. Il offre une interface conviviale et une intégration transparente avec l'écosystème Proton.

## Site Web / Lien
- [Site Officiel](https://proton.me/pass)
- [GitHub (Applications)](https://github.com/protonprivacy)

## Installation
Disponible en tant qu'extension de navigateur (Chrome, Firefox, Brave, Edge) et application mobile (iOS, Android).
```bash
# Installez via le magasin d'extensions de votre navigateur ou le magasin d'applications mobile.
```

## Commandes Utiles / Astuces
- **Hide-my-email** : Créez des alias d'email uniques pour chaque service afin de réduire le spam et protéger votre véritable adresse email.
- **Notes sécurisées** : Stockez des informations sensibles (codes PIN, clés de récupération, coordonnées bancaires) dans les notes chiffrées.
- **2FA intégré** : Générez des codes d'authentification à deux facteurs directement dans l'application pour des connexions plus rapides.
- **Coffres (Vaults)** : Organisez vos identifiants et notes dans différents coffres (ex: Personnel, Travail) pour une meilleure gestion.

## Configuration
- **Synchronisation** : Se synchronise automatiquement sur tous les appareils connectés au compte Proton.
- **Importation** : Permet d'importer facilement des mots de passe depuis d'autres navigateurs (Chrome, Firefox) ou gestionnaires de mots de passe (LastPass, 1Password, Bitwarden).
- **Verrouillage automatique** : Configurez l'application pour qu'elle se verrouille automatiquement après une période d'inactivité.

## Notes
- Parfait pour ceux qui sont déjà dans l'écosystème Proton.
- La fonction "Hide-my-email" est particulièrement utile pour s'inscrire à des newsletters ou des services temporaires sans exposer son email principal.
- Il est open-source et audité, garantissant transparence et sécurité.

# Figma

> [!TIP]

## Description
Figma is a cloud-based interface design tool that enables teams to create, test, and deliver better designs from start to finish. Known for its real-time collaboration capabilities, it has become an industry standard for UI/UX design.

## Website / Link
- [Official Website](https://www.figma.com/)
- [Downloads (Desktop App)](https://www.figma.com/downloads/)

## Installation
Figma is primarily web-based, but a desktop application is available for better performance and font management.

```bash
# Download the installer from the official website
# Windows: https://www.figma.com/download/desktop/win
# macOS: https://www.figma.com/download/desktop/mac
```

## Useful Commands / Tips
- `A` : Select Artboard / Frame tool
- `R` : Rectangle tool
- `T` : Text tool
- `Shift + A` : Add Auto Layout (Essential for responsive design)
- `Ctrl + Alt + K` (Windows) / `Cmd + Option + K` (Mac) : Create Component
- `Ctrl + /` (Windows) / `Cmd + /` (Mac) : Quick Actions / Search

## Configuration
- **Font Helper**: If using the web version, install the "Figma Agent" to access local fonts.
- **Preferences**: Accessible via the Figma menu (top left) to toggle rulers, pixel grid, etc.

## Notes
- **Real-time Collaboration**: Multiple users can edit the same file simultaneously.
- **Dev Mode**: A dedicated space for developers to inspect code and export assets (requires a paid plan for full features).
- **Plugins**: Extensive ecosystem of community plugins to automate tasks.

# Version Française

## Description
Figma est un outil de design d'interface collaboratif basé sur le cloud. Il permet aux équipes de créer, tester et livrer de meilleures conceptions du début à la fin. Connu pour ses capacités de collaboration en temps réel, il est devenu un standard de l'industrie pour l'UI/UX design.

## Site Web / Lien
- [Site Officiel](https://www.figma.com/)
- [Téléchargements (App Bureau)](https://www.figma.com/downloads/)

## Installation
Figma est principalement basé sur le web, mais une application de bureau est disponible pour de meilleures performances et la gestion des polices locales.

```bash
# Télécharger l'installateur depuis le site officiel
# Windows: https://www.figma.com/download/desktop/win
# macOS: https://www.figma.com/download/desktop/mac
```

## Commandes Utiles / Astuces
- `A` : Outil Plan de travail (Frame)
- `R` : Outil Rectangle
- `T` : Outil Texte
- `Shift + A` : Ajouter un Auto Layout (Essentiel pour le design réactif)
- `Ctrl + Alt + K` (Windows) / `Cmd + Option + K` (Mac) : Créer un Composant
- `Ctrl + /` (Windows) / `Cmd + /` (Mac) : Actions rapides / Recherche

## Configuration
- **Gestionnaire de polices** : Si vous utilisez la version web, installez le "Figma Agent" pour accéder à vos polices locales.
- **Préférences** : Accessible via le menu Figma (en haut à gauche) pour afficher les règles, la grille de pixels, etc.

## Notes
- **Collaboration en temps réel** : Travaillez simultanément sur le même fichier avec votre équipe.
- **Dev Mode** : Espace dédié aux développeurs pour inspecter le code et exporter les assets (nécessite un plan payant pour toutes les fonctionnalités).
- **Plugins** : Accès à des milliers de ressources et extensions gratuites pour automatiser les tâches.

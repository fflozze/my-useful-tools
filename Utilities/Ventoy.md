# Ventoy

> [!TIP]
> [Passer à la version française](#version-française)

## Description
Ventoy is an open-source tool to create bootable USB drives for ISO/WIM/IMG/VHD(X)/EFI files. With Ventoy, you don't need to format the disk over and over, you just need to copy the files to the USB drive and boot them directly.

## Website / Link
- [Official Website](https://www.ventoy.net/)
- [GitHub Repository](https://github.com/ventoy/Ventoy)

## Installation
```bash
# Download the package for Windows or Linux.
# Run Ventoy2Disk.exe (Windows) or use the script (Linux) to install it on your USB drive.
```

## Useful Commands / Tips
- Simply copy multiple ISO files to the USB drive; Ventoy will give you a boot menu to select them.
- You can create a `ventoy` folder on the USB and put a `ventoy.json` file for customization (themes, persistence, etc.).

## Configuration
- Persistent storage can be configured for some Linux distributions using the `CreatePersistentImg.sh` script provided in the Ventoy package.

## Notes
A must-have tool for any technician or IT enthusiast. It supports both Legacy BIOS and UEFI.

# Version Française

## Description
Ventoy est un outil open-source permettant de créer des clés USB bootables pour les fichiers ISO/WIM/IMG/VHD(X)/EFI. Avec Ventoy, vous n'avez pas besoin de formater le disque encore et encore, il vous suffit de copier les fichiers sur la clé USB et de les démarrer directement.

## Site Web / Lien
- [Site Officiel](https://www.ventoy.net/)
- [Dépôt GitHub](https://github.com/ventoy/Ventoy)

## Installation
```bash
# Téléchargez le pack pour Windows ou Linux.
# Lancez Ventoy2Disk.exe (Windows) ou utilisez le script (Linux) pour l'installer sur votre clé USB.
```

## Commandes Utiles / Astuces
- Copiez simplement plusieurs fichiers ISO sur la clé USB ; Ventoy vous affichera un menu au démarrage pour les sélectionner.
- Vous pouvez créer un dossier `ventoy` sur la clé et y placer un fichier `ventoy.json` pour la personnalisation (thèmes, persistance, etc.).

## Configuration
- Le stockage persistant peut être configuré pour certaines distributions Linux en utilisant le script `CreatePersistentImg.sh` fourni dans le pack Ventoy.

## Notes
Un outil indispensable pour tout technicien ou passionné d'informatique. Il supporte à la fois le BIOS Legacy et l'UEFI.

# Parrot Security OS

> [!TIP]
> [Passer à la version française](#version-française)

## Description
Parrot Security OS is a Linux distribution based on Debian with a focus on security, privacy, and development. It includes a full arsenal of security-oriented tools for penetration testing, digital forensics, and reverse engineering.

## Website / Link
- [Official Website](https://www.parrotsec.org/)
- [Documentation](https://doc.parrotsec.org/)

## Installation
```bash
# Download the ISO from the official website and create a bootable USB
# or install via architect ISO for a custom setup
```

## Useful Commands / Tips
- `sudo apt update && sudo apt full-upgrade`: Update the system
- `parrot-menu`: Access the security tools menu
- `anonsurf start`: Start anonymous surfing mode

## Configuration
Default credentials for live mode are usually `parrot` / `parrot`.

## Notes
Excellent alternative to Kali Linux, often considered lighter and more user-friendly for daily use.

# Version Française

## Description
Parrot Security OS est une distribution Linux basée sur Debian axée sur la sécurité, la vie privée et le développement. Elle comprend un arsenal complet d'outils orientés sécurité pour les tests d'intrusion, l'informatique légale (forensics) et l'ingénierie inverse.

## Site Web / Lien
- [Site Officiel](https://www.parrotsec.org/)
- [Documentation](https://doc.parrotsec.org/)

## Installation
```bash
# Téléchargez l'ISO depuis le site officiel et créez une clé USB bootable
# ou installez via l'ISO architect pour une configuration personnalisée
```

## Commandes Utiles / Astuces
- `sudo apt update && sudo apt full-upgrade` : Mettre à jour le système
- `parrot-menu` : Accéder au menu des outils de sécurité
- `anonsurf start` : Activer le mode de navigation anonyme

## Configuration
Les identifiants par défaut pour le mode live sont généralement `parrot` / `parrot`.

## Notes
Excellente alternative à Kali Linux, souvent considérée comme plus légère et plus conviviale pour un usage quotidien.

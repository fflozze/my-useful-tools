# Linux Mint

> [!TIP]
> [Passer à la version française](#version-française)

## Description
Linux Mint is a modern, elegant, and comfortable GNU/Linux desktop distribution which is both powerful and easy to use. It is based on Ubuntu and Debian.

## Website / Link
- [Official Website](https://linuxmint.com/)
- [Forums](https://forums.linuxmint.com/)

## Installation
```bash
# Download the ISO (Cinnamon, MATE, or Xfce edition)
# Create a bootable USB and follow the graphical installer
```

## Useful Commands / Tips
- `sudo apt update`: Update package list
- `sudo apt upgrade`: Apply updates
- `mintupdate`: Launch the Update Manager (GUI)

## Configuration
- Update Manager: Use it to manage kernel versions and security updates easily.
- Software Sources: Select the fastest mirrors for your location.

## Notes
The Cinnamon edition is the most popular and provides a very familiar interface for Windows users.

# Version Française

## Description
Linux Mint est une distribution GNU/Linux moderne, élégante et confortable, à la fois puissante et facile à utiliser. Elle est basée sur Ubuntu et Debian.

## Site Web / Lien
- [Site Officiel](https://linuxmint.com/)
- [Forums](https://forums.linuxmint.com/)

## Installation
```bash
# Téléchargez l'ISO (édition Cinnamon, MATE ou Xfce)
# Créez une clé USB bootable et suivez l'installateur graphique
```

## Commandes Utiles / Astuces
- `sudo apt update` : Mettre à jour la liste des paquets
- `sudo apt upgrade` : Appliquer les mises à jour
- `mintupdate` : Lancer le gestionnaire de mises à jour (interface graphique)

## Configuration
- Gestionnaire de mises à jour : Utilisez-le pour gérer facilement les versions du noyau et les mises à jour de sécurité.
- Sources de logiciels : Sélectionnez les miroirs les plus rapides pour votre localisation.

## Notes
L'édition Cinnamon est la plus populaire et offre une interface très familière pour les utilisateurs de Windows.

# PhpStorm

> [!TIP]
> [Passer à la version française](#version-française)

## Description
PhpStorm is a cross-platform Integrated Development Environment (IDE) for PHP, built by JetBrains. It provides an intelligent editor for PHP, HTML, and JavaScript with on-the-fly code analysis, error prevention, and automated refactorings. It offers deep code understanding and top-notch coding assistance for all major PHP frameworks such as Symfony, Laravel, Drupal, WordPress, and more.

## Website / Link
- [Official Website](https://www.jetbrains.com/phpstorm/)
- [Documentation](https://www.jetbrains.com/help/phpstorm/)

## Installation
Download from the official website or use the JetBrains Toolbox App.

```bash
# Via Winget (Windows)
winget install JetBrains.PhpStorm
```

## Useful Commands / Tips
- `Double Shift`: Search everywhere (files, classes, symbols, actions).
- `Ctrl + Shift + A`: Find Action (helps you find commands without memorizing shortcuts).
- `Alt + Enter`: Show Intention Actions and Quick Fixes.
- `Ctrl + Click` / `Ctrl + B`: Go to declaration.
- `Ctrl + Alt + L`: Reformat Code.
- `Shift + F6`: Rename refactoring.

## Configuration
- **PHP Interpreter**: specific via `File > Settings > Languages & Frameworks > PHP`.
- **Composer**: `File > Settings > Languages & Frameworks > PHP > Composer`.
- **Plugins**: Accessible via `File > Settings > Plugins`.

## Notes
- Includes full Database/SQL support (DataGrip features).
- Excellent testing integration (PHPUnit, Pest).
- Integration with Version Control Systems (Git, GitHub).
- Requires a subscription (free for students/open source maintainers).

# Version Française

## Description
PhpStorm est un environnement de développement intégré (IDE) multiplateforme pour PHP, conçu par JetBrains. Il fournit un éditeur intelligent pour PHP, HTML et JavaScript avec analyse de code à la volée, prévention des erreurs et refactorisations automatisées. Il offre une compréhension approfondie du code et une assistance au codage de premier ordre pour tous les principaux frameworks PHP tels que Symfony, Laravel, Drupal, WordPress, et plus encore.

## Site Web / Lien
- [Site Officiel](https://www.jetbrains.com/phpstorm/)
- [Documentation](https://www.jetbrains.com/help/phpstorm/)

## Installation
Télécharger depuis le site officiel ou utiliser l'application JetBrains Toolbox.

```bash
# Via Winget (Windows)
winget install JetBrains.PhpStorm
```

## Commandes Utiles / Astuces
- `Double Shift` : Rechercher partout (fichiers, classes, symboles, actions).
- `Ctrl + Shift + A` : Trouver une action (aide à trouver des commandes sans mémoriser les raccourcis).
- `Alt + Enter` : Afficher les actions d'intention et les corrections rapides.
- `Ctrl + Click` / `Ctrl + B` : Aller à la déclaration.
- `Ctrl + Alt + L` : Reformater le code.
- `Shift + F6` : Renommer (Refactoring).

## Configuration
- **Interpréteur PHP** : Spécifique via `File > Settings > Languages & Frameworks > PHP`.
- **Composer** : `File > Settings > Languages & Frameworks > PHP > Composer`.
- **Plugins** : Accessible via `File > Settings > Plugins`.

## Notes
- Inclut le support complet Base de données/SQL (fonctionnalités DataGrip).
- Excellente intégration des tests (PHPUnit, Pest).
- Intégration avec les systèmes de contrôle de version (Git, GitHub).
- Nécessite un abonnement (gratuit pour les étudiants/mainteneurs open source).

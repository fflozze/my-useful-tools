# Google Gemini

> [!TIP]
> [Passer à la version française](#version-française)

## Description
Google Gemini is a family of multimodal AI models developed by Google DeepMind. It is designed to understand and generate text, code, images, audio, and video. It serves as a direct competitor to OpenAI's GPT models and is deeply integrated into the Google ecosystem.

## Website / Link
- [Official Website](https://gemini.google.com/)
- [Google DeepMind Gemini Page](https://deepmind.google/technologies/gemini/)

## Key Features
- **Multimodal Capabilities**: Native understanding of text, images, audio, video, and code from the ground up.
- **Google Workspace Integration**: Can access and summarize information from Google Drive, Docs, Sheets, Gmail, etc. (with extensions enabled).
- **Coding**: Capable of writing, explaining, and debugging code in various languages.
- **Large Context Window**: Processing of very large documents and codebases (especially in the 1.5 Pro architecture).

## Configuration
- **Extensions**: Enable/Disable extensions for Google Flights, Hotels, Maps, Workspace, and YouTube in settings for personalized assistance.
- **Theme**: Light/Dark mode support available in settings.

## Notes
- "Gemini Advanced" is the paid tier offering access to the most capable models (e.g., 1.5 Pro, Ultra).
- The model is continuously updated, with different versions (Nano for on-device, Flash for speed, Pro for scaling, Ultra for max complexity) optimized for different use cases.

# Version Française

## Description
Google Gemini est une famille de modèles d'IA multimodaux développée par Google DeepMind. Il est conçu pour comprendre et générer du texte, du code, des images, de l'audio et de la vidéo. Il sert de concurrent direct aux modèles GPT d'OpenAI et est profondément intégré à l'écosystème Google.

## Site Web / Lien
- [Site Officiel](https://gemini.google.com/)
- [Page Google DeepMind Gemini](https://deepmind.google/technologies/gemini/)

## Fonctionnalités Clés
- **Capacités Multimodales** : Compréhension native du texte, des images, de l'audio, de la vidéo et du code dès la conception.
- **Intégration Google Workspace** : Peut accéder aux informations de Google Drive, Docs, Sheets, Gmail, etc. et les résumer (avec les extensions activées).
- **Codage** : Capable d'écrire, d'expliquer et de déboguer du code dans divers langages.
- **Grande Fenêtre de Contexte** : Traitement de très grands documents et bases de code (particulièrement dans l'architecture 1.5 Pro).

## Configuration
- **Extensions** : Activez/désactivez les extensions pour Google Flights, Hotels, Maps, Workspace et YouTube dans les paramètres pour une assistance personnalisée.
- **Thème** : Support du mode Clair/Sombre disponible dans les paramètres.

## Notes
- "Gemini Advanced" est l'offre payante donnant accès aux modèles les plus performants (ex: 1.5 Pro, Ultra).
- Le modèle est continuellement mis à jour, avec différentes versions (Nano pour mobile, Flash pour la vitesse, Pro pour l'échelle, Ultra pour la complexité maximale) optimisées pour différents cas d'usage.

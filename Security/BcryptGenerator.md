# Bcrypt Generator

> [!TIP]
> [Passer à la version française](#version-française)

## Description
[Bcrypt Generator](https://bcrypt-generator.com/) is a simple online tool that allows you to generate Bcrypt hashes from string inputs and verify if a string matches a Bcrypt hash. It is useful for developers who need to quickly generate hashes for passwords or test authentication flows.

## Website / Link
- [Official Website](https://bcrypt-generator.com/)

## Installation
No installation required, it is a web-based tool.

## Useful Commands / Tips
- **Generate Hash**: Convert any string into a Bcrypt hash using the "Encrypt" function.
- **Check Hash**: Verify if a plain text string matches a given Bcrypt hash using the "Check" function.
- **Adjustable Rounds**: You can specify the number of rounds (work factor) for hashing. The default is often 12. A higher number means more security but takes longer to compute.

## Configuration
- **Rounds**: Choose the complexity of the encryption (between 4 and 31).

## Notes
- Bcrypt is a password-hashing function designed to be slow, making it resistant to brute-force search attacks.
- **Security Warning**: Never use online tools for real, sensitive production passwords. Use them only for testing or development data to avoid exposing real credentials.

# Version Française

## Description
[Bcrypt Generator](https://bcrypt-generator.com/) est un outil en ligne simple qui vous permet de générer des hachages Bcrypt à partir de chaînes de caractères et de vérifier si une chaîne correspond à un hachage Bcrypt. Il est utile pour les développeurs qui ont besoin de générer rapidement des hachages pour des mots de passe ou de tester des flux d'authentification.

## Site Web / Lien
- [Site Officiel](https://bcrypt-generator.com/)

## Installation
Aucune installation requise, c'est un outil basé sur le web.

## Commandes Utiles / Astuces
- **Générer un hash** : Convertissez n'importe quelle chaîne en hash Bcrypt en utilisant la fonction "Encrypt".
- **Vérifier un hash** : Vérifiez si un texte brut correspond à un hash Bcrypt donné en utilisant la fonction "Check".
- **Tours (Rounds) Ajustables** : Vous pouvez spécifier le nombre de tours (facteur de travail) pour le hachage. La valeur par défaut est souvent 12. Un nombre plus élevé signifie plus de sécurité mais prend plus de temps à calculer.

## Configuration
- **Rounds** : Choisissez la complexité du chiffrement (entre 4 et 31).

## Notes
- Bcrypt est une fonction de hachage de mot de passe conçue pour être lente, ce qui la rend résistante aux attaques par force brute.
- **Avertissement de sécurité** : N'utilisez jamais d'outils en ligne pour de vrais mots de passe de production sensibles. Utilisez-les uniquement pour des données de test ou de développement afin d'éviter d'exposer de vrais identifiants.

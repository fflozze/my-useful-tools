# PyCharm (JetBrains)

> [!TIP]
> [Passer à la version française](#version-française)

## Description
PyCharm is an Integrated Development Environment (IDE) for the Python language, developed by JetBrains. It offers intelligent code analysis, a powerful graphical debugger, an integrated unit tester, and excellent integration with version control systems (Git, etc.). It is designed to improve Python developer productivity, with specific support for web development (Django, Flask) and data science.

## Website / Link
- [Official Website](https://www.jetbrains.com/pycharm/)
- [Documentation](https://www.jetbrains.com/help/pycharm/)

## Installation
Via JetBrains Toolbox (Recommended) or direct download:
```bash
# Download and install JetBrains Toolbox App to manage JetBrains tools
https://www.jetbrains.com/toolbox-app/
```

## Useful Commands / Tips
- `Double Shift`: Search Everywhere.
- `Alt + Enter`: Show Context Actions and Quick Fixes.
- `Ctrl + Click` (or `Ctrl + B`): Go to Definition/Declaration.
- `Ctrl + Alt + L`: Reformat Code (PEP 8 compliance).
- `Shift + F6`: Rename Symbol (Smart Refactoring).
- `Ctrl + Shift + F`: Find in Files.
- **Debug**: Use the visual debugger to set breakpoints and inspect variables in real-time.

## Configuration
- **Python Interpreter**: Critical to configure correctly (Virtualenv, Conda, System) via `Settings` > `Project` > `Python Interpreter`.
- **Requirements**: PyCharm automatically detects `requirements.txt` files and suggests installing missing dependencies.

## Notes
- **Editions**:
    - **Community**: Free and Open Source. Sufficient for pure Python and basic scientific development.
    - **Professional**: Paid. Adds support for Web development (Django, Flask, FastAPI), JavaScript/HTML/CSS, SQL/Databases, and advanced Data Science features (Jupyter, etc.).
- Shares the same base as WebStorm and IntelliJ IDEA, so shortcuts and interface are familiar if you use other JetBrains tools.

# Version Française

## Description
PyCharm est un environnement de développement intégré (IDE) pour le langage Python, développé par JetBrains. Il offre une analyse de code intelligente, un débogueur graphique puissant, un testeur unitaire intégré, et une excellente intégration avec les systèmes de contrôle de version (Git, etc.). Il est conçu pour améliorer la productivité des développeurs Python, avec un support spécifique pour le développement web (Django, Flask) et la science des données.

## Site Web / Lien
- [Site Officiel](https://www.jetbrains.com/fr-fr/pycharm/)
- [Documentation](https://www.jetbrains.com/help/pycharm/)

## Installation
Via JetBrains Toolbox (Recommandé) ou téléchargement direct :
```bash
# Télécharger et installer JetBrains Toolbox App pour gérer les outils JetBrains
https://www.jetbrains.com/toolbox-app/
```

## Commandes Utiles / Astuces
- `Double Shift` : Rechercher partout (Search Everywhere).
- `Alt + Enter` : Afficher les actions contextuelles et les corrections rapides (Quick fixes).
- `Ctrl + Click` (ou `Ctrl + B`) : Aller à la définition/déclaration.
- `Ctrl + Alt + L` : Reformater le code selon les standards PEP 8.
- `Shift + F6` : Renommer un symbole (Refactoring intelligent).
- `Ctrl + Shift + F` : Rechercher une chaîne de caractères dans tout le projet.
- **Debug** : Utiliser le débogueur visuel pour mettre des points d'arrêt et inspecter les variables en temps réel.

## Configuration
- **Interpréteur Python** : Il est crucial de configurer correctement l'interpréteur (Virtualenv, Conda, System) via `Settings` > `Project` > `Python Interpreter`.
- **Requirements** : PyCharm détecte automatiquement les fichiers `requirements.txt` et propose d'installer les dépendances manquantes.

## Notes
- **Éditions** :
    - **Community** : Gratuite et Open Source. Suffisante pour le développement Python pur et scientifique de base.
    - **Professional** : Payante. Ajoute le support pour le développement Web (Django, Flask, FastAPI), JavaScript/HTML/CSS, SQL/Databases, et des fonctionnalités avancées pour la Data Science (Jupyter, etc.).
- Il partage la même base que WebStorm et IntelliJ IDEA, donc les raccourcis et l'interface sont familiers si vous utilisez d'autres outils JetBrains.

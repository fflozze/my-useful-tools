# Proton Mail

> [!TIP]
> [Passer à la version française](#version-française)

## Description
Proton Mail is a secure email service based in Switzerland. It stands out for its automatic end-to-end encryption, ensuring that no one else but you (not even Proton) can read your emails. It respects privacy and does not sell your data.

## Website / Link
- [Official Website](https://proton.me/mail)
- [Proton Bridge (for desktop clients)](https://proton.me/mail/bridge)
- [Tor Onion Site](https://protonmailrmez3lotccipshtkleegetolb73fuirgj7r4o4vfu7ozyd.onion/)

## Installation
Proton Mail is primarily a web service, but applications are available:
- **Web**: Accessible via any browser.
- **Mobile**: iOS and Android apps available on stores.
- **Desktop (via Bridge)**: To use Proton Mail with Thunderbird, Outlook, or Apple Mail, install **Proton Bridge**.

## Useful Commands / Tips
- **Proton Bridge**: Connects your encrypted account to a standard mail client (IMAP/SMTP).
- **Aliases (SimpleLogin)**: Use aliases to hide your real email address when signing up for third-party services.
- **Keyboard Shortcuts**: Press `?` in the web interface to view available shortcuts.
- **Proton Drive**: A Proton account also includes secure storage (Drive), a VPN, and a Calendar.
- **Encryption**: Send password-protected encrypted emails to non-Proton users.

## Configuration
- **2FA**: It is strongly recommended to enable two-factor authentication in security settings.
- **Custom Domain**: With a paid subscription, you can use your own domain name (e.g., @mycompany.com).
- **Filters/Sieve**: Advanced filtering options to organize your inbox automatically.

## Notes
- The free version is very functional but limited in storage and features.
- Data is hosted in Switzerland, benefiting from strict privacy laws.
- No ads, no tracking.

# Version Française

## Description
Proton Mail est un service de messagerie électronique sécurisé basé en Suisse. Il se distingue par son chiffrement de bout en bout automatique, garantissant que personne d'autre que vous (pas même Proton) ne peut lire vos e-mails. Il respecte la vie privée et ne vend pas vos données.

## Site Web / Lien
- [Site Officiel](https://proton.me/mail)
- [Proton Bridge (pour clients desktop)](https://proton.me/mail/bridge)

## Installation
Proton Mail est principalement un service web, mais des applications sont disponibles :
- **Web** : Accessible via n'importe quel navigateur.
- **Mobile** : Applications iOS et Android disponibles sur les stores.
- **Desktop (via Bridge)** : Pour utiliser Proton Mail avec Thunderbird, Outlook ou Apple Mail, installez **Proton Bridge**.

## Commandes Utiles / Astuces
- **Proton Bridge** : Permet de connecter votre compte chiffré à un client mail standard (IMAP/SMTP).
- **Alias (SimpleLogin)** : Utilisez des alias pour masquer votre véritable adresse e-mail lors de l'inscription à des services tiers.
- **Raccourcis clavier** : Appuyez sur `?` dans l'interface web pour voir les raccourcis disponibles.
- **Proton Drive** : Un compte Proton inclut également un espace de stockage sécurisé (Drive), un VPN et un calendrier.
- **Chiffrement** : Envoyez des e-mails chiffrés protégés par mot de passe à des utilisateurs non-Proton.

## Configuration
- **2FA** : Il est fortement recommandé d'activer l'authentification à deux facteurs dans les paramètres de sécurité.
- **Domaine personnalisé** : Avec un abonnement payant, vous pouvez utiliser votre propre nom de domaine (@monentreprise.com).
- **Filtres** : Options de filtrage avancées pour organiser votre boîte de réception automatiquement.

## Notes
- La version gratuite est très fonctionnelle mais limitée en stockage et fonctionnalités.
- Les données sont hébergées en Suisse, bénéficiant de lois strictes sur la protection de la vie privée.
- Pas de publicités, pas de traçage.

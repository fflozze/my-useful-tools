# React

> [!TIP]
> [Passer à la version française](#version-française)

## Description
React is a free and open-source front-end JavaScript library for building user interfaces based on UI components. It is maintained by Meta (formerly Facebook) and a community of individual developers and companies. React allows developers to create large web applications that can change data, without reloading the page. Its main goal is to be fast, scalable, and simple.

## Website / Link
- [Official Website](https://react.dev/)
- [Repository](https://github.com/facebook/react)

## Installation
```bash
# Install React
npm install react
```

## Useful Commands / Tips
- React is typically installed with a framework like Next.js or a build tool like Vite.
- `npx create-react-app my-app`: (Legacy) Command to create a new React application.
- `npm create vite@latest`: Modern way to scaffold a React project.

## Configuration
- React itself doesn't have a specific configuration file, but it depends on the build tool configuration (e.g., `vite.config.js`, `next.config.js`).

## Notes
React introduced the concept of the Virtual DOM and JSX, revolutionizing how we think about UI development.

# Version Française

## Description
React est une bibliothèque JavaScript front-end libre et open-source pour la création d'interfaces utilisateur basées sur des composants UI. Elle est maintenue par Meta (anciennement Facebook) et une communauté de développeurs individuels et d'entreprises. React permet aux développeurs de créer de grandes applications web capables de modifier des données sans recharger la page. Son objectif principal est d'être rapide, évolutif et simple.

## Site Web / Lien
- [Site Officiel](https://react.dev/)
- [Dépôt](https://github.com/facebook/react)

## Installation
```bash
# Installer React
npm install react
```

## Commandes Utiles / Astuces
- React est généralement installé avec un framework comme Next.js ou un outil de build comme Vite.
- `npx create-react-app my-app`: (Ancien) Commande pour créer une nouvelle application React.
- `npm create vite@latest`: Manière moderne d'échafauder un projet React.

## Configuration
- React en lui-même n'a pas de fichier de configuration spécifique, mais dépend de la configuration de l'outil de build (par exemple, `vite.config.js`, `next.config.js`).

## Notes
React a introduit le concept de DOM virtuel et de JSX, révolutionnant la façon dont nous concevons le développement d'interfaces utilisateur.

# Tiny11

> [!TIP]
> [Passer à la version française](#version-française)

## Description
Tiny11 is a stripped-down, bloatware-free version of Windows 11. It is designed to run on systems with lower hardware specifications (as low as 2GB of RAM) by removing non-essential components.

## Website / Link
- [Archive.org (NTDEV)](https://archive.org/details/tiny-11-NTDEV)

## Installation
```bash
# Download the ISO from the official source on Archive.org.
# Create a bootable USB and install like a regular Windows 11 version.
```

## Useful Commands / Tips
- Tiny11 still requires a valid Windows 11 license.
- It removes many telemetry and background services, which improves performance on older hardware.

## Configuration
- You can still install most software from the Microsoft Store or via traditional installers.

## Notes
Use with caution as some components you might need could have been removed. Always back up your data.

# Version Française

## Description
Tiny11 est une version allégée et débarrassée des logiciels superflus (bloatware) de Windows 11. Elle est conçue pour fonctionner sur des systèmes avec des spécifications matérielles plus faibles (jusqu'à 2 Go de RAM) en supprimant les composants non essentiels.

## Site Web / Lien
- [Archive.org (NTDEV)](https://archive.org/details/tiny-11-NTDEV)

## Installation
```bash
# Téléchargez l'ISO depuis la source officielle sur Archive.org.
# Créez une clé USB bootable et installez comme une version normale de Windows 11.
```

## Commandes Utiles / Astuces
- Tiny11 nécessite toujours une licence Windows 11 valide.
- De nombreux services de télémétrie et d'arrière-plan ont été supprimés, ce qui améliore les performances sur le matériel ancien.

## Configuration
- Vous pouvez toujours installer la plupart des logiciels via le Microsoft Store ou les installateurs traditionnels.

## Notes
À utiliser avec précaution car certains composants dont vous pourriez avoir besoin peuvent avoir été supprimés. Sauvegardez toujours vos données.

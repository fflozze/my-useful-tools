# FontForge

> [!TIP]
> [Passer à la version française](#version-française)

## Description
FontForge is the reference open-source font editor. It allows you to create, edit, and convert fonts in all common formats (TrueType, OpenType, PostScript, etc.).

## Website / Link
- [Official Website](https://fontforge.org/)
- [Repository](https://github.com/fontforge/fontforge)

## Installation

### Windows & Mac
Download the installer from the official website:
- [Windows Download](https://fontforge.org/en-US/downloads/windows-dl/)
- [Mac Download](https://fontforge.org/en-US/downloads/mac-dl/)

### Linux
Often available in official repositories:
```bash
sudo apt install fontforge  # Debian/Ubuntu
sudo dnf install fontforge  # Fedora
```

## Useful Commands / Tips
- **Import SVG**: You can import vector drawings in `.svg` format to transform them directly into glyphs.
- **Bezier Curves**: Powerful tool to draw and adjust character curves with precision.
- **Kerning**: Advanced management of spacing between letters (kerning).

## Configuration
No specific configuration required for basic usage.

# Version Française

## Description
C'est l'éditeur de polices de caractères open-source de référence. Il permet de créer, d'éditer et de convertir des polices dans tous les formats courants (TrueType, OpenType, PostScript, etc.).

## Site Web / Lien
- [Site Officiel](https://fontforge.org/)
- [Dépôt](https://github.com/fontforge/fontforge)

## Installation
### Windows & Mac
Télécharger l'installateur depuis le site officiel :
- [Windows Download](https://fontforge.org/en-US/downloads/windows-dl/)
- [Mac Download](https://fontforge.org/en-US/downloads/mac-dl/)

### Linux
Souvent disponible dans les dépôts officiels :
```bash
sudo apt install fontforge  # Debian/Ubuntu
sudo dnf install fontforge  # Fedora
```

## Commandes Utiles / Astuces
- **Import SVG** : Vous pouvez importer des dessins vectoriels au format `.svg` pour les transformer directement en glyphes.
- **Courbes de Bézier** : Outil puissant pour dessiner et ajuster les courbes des caractères avec précision.
- **Kerning** : Gestion avancée de l'espacement entre les lettres (crénage).

## Configuration
Pas de configuration spécifique requise pour l'utilisation de base.

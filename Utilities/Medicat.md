# Médicat USB

> [!TIP]
> [Passer à la version française](#version-française)

## Description
Médicat USB is a multi-boot bootable USB toolkit, intended to be a modern successor to Hiren's BootCD. It contains a collection of tools for troubleshooting, repairing, and maintaining computers.

## Website / Link
- [Official Website](https://medicatusb.com/)
- [Download Page](https://medicatusb.com/download/)

## Installation
```bash
# It is recommended to use Ventoy to install Médicat.
# Download the Médicat files and copy them to your Ventoy USB drive.
```

## Useful Commands / Tips
- Includes a mini-Windows (WinPE) with many pre-installed tools for file recovery, partition management, and password resetting.
- Contains Linux-based tools for deep system scanning.

## Configuration
- Follow the official "How to Install" guide on the website, as it may involve multiple parts or a specific installer.

## Notes
A very heavy toolkit (several GBs) but extremely powerful for PC repair.

# Version Française

## Description
Médicat USB est un kit d'outils multi-boot sur clé USB, conçu pour être le successeur moderne de Hiren's BootCD. Il contient une collection d'outils pour le dépannage, la réparation et la maintenance des ordinateurs.

## Site Web / Lien
- [Site Officiel](https://medicatusb.com/)
- [Page de Téléchargement](https://medicatusb.com/download/)

## Installation
```bash
# Il est recommandé d'utiliser Ventoy pour installer Médicat.
# Téléchargez les fichiers Médicat et copiez-les sur votre clé USB Ventoy.
```

## Commandes Utiles / Astuces
- Comprend un mini-Windows (WinPE) avec de nombreux outils pré-installés pour la récupération de fichiers, la gestion de partitions et la réinitialisation de mots de passe.
- Contient des outils basés sur Linux pour une analyse approfondie du système.

## Configuration
- Suivez le guide officiel "How to Install" sur le site, car cela peut impliquer plusieurs parties ou un installateur spécifique.

## Notes
Un kit d'outils très lourd (plusieurs Go) mais extrêmement puissant pour la réparation de PC.

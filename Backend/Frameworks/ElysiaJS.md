# ElysiaJS

> [!TIP]
> [Passer à la version française](#version-française)

## Description
ElysiaJS is a fast and ergonomic web framework for building backend servers, optimized for the Bun runtime. It prioritizes developer experience with end-to-end type safety (using TypeScript) and high performance. It allows you to write type-safe APIs without the complexity of manual type definitions.

## Website / Link
- [Official Website](https://elysiajs.com/)
- [Repository](https://github.com/elysiajs/elysia)

## Installation
```bash
# Create a new Elysia app using Bun
bun create elysia app
cd app
bun dev
```

## Useful Commands / Tips
- `bun dev`: Starts the development server.
- Elysia uses a plugin system for many features (e.g., Swagger, JWT).

## Configuration
- Configuration is often done directly in the main application file (e.g., `src/index.ts`) when instantiating the `Elysia` class.

## Notes
Elysia is one of the main reasons to use Bun for backend development, offering speed and a syntax similar to Express but modernized.

# Version Française

## Description
ElysiaJS est un framework web rapide et ergonomique pour la création de serveurs backend, optimisé pour le runtime Bun. Il privilégie l'expérience développeur avec une sécurité de type de bout en bout (utilisant TypeScript) et de hautes performances. Il vous permet d'écrire des API sécurisées sans la complexité des définitions de type manuelles.

## Site Web / Lien
- [Site Officiel](https://elysiajs.com/)
- [Dépôt](https://github.com/elysiajs/elysia)

## Installation
```bash
# Créer une nouvelle application Elysia avec Bun
bun create elysia app
cd app
bun dev
```

## Commandes Utiles / Astuces
- `bun dev`: Démarre le serveur de développement.
- Elysia utilise un système de plugins pour de nombreuses fonctionnalités (par exemple, Swagger, JWT).

## Configuration
- La configuration se fait souvent directement dans le fichier principal de l'application (par exemple, `src/index.ts`) lors de l'instanciation de la classe `Elysia`.

## Notes
Elysia est l'une des principales raisons d'utiliser Bun pour le développement backend, offrant vitesse et une syntaxe similaire à Express mais modernisée.

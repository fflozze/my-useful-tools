# i18n (i18next)

> [!TIP]
> [Passer à la version française](#version-française)

## Description
i18next is a very popular internationalization framework for browsers or any other JavaScript environment (like Node.js). It provides a complete solution to localize your product from web to mobile and desktop. It is often used with `react-i18next` for React applications.

## Website / Link
- [Official Website](https://www.i18next.com/)
- [Repository](https://github.com/i18next/i18next)
- [React Extension](https://react.i18next.com/)

## Installation
```bash
# Install i18next and the React extension
npm install i18next react-i18next
```

## Useful Commands / Tips
- `t('key')`: Standard function to translate a key.
- `i18n.changeLanguage('fr')`: Method to change the current language dynamically.

## Configuration
- Typically initialized in a dedicated file (e.g., `i18n.js`) where you configure resources (translations), the default language, and interpolation options.

## Notes
i18next goes beyond just standard i18n features; it includes support for pluralization, context, and interpolation.

# Version Française

## Description
i18next est un framework d'internationalisation très populaire pour les navigateurs ou tout autre environnement JavaScript (comme Node.js). Il fournit une solution complète pour localiser votre produit, du web au mobile et au bureau. Il est souvent utilisé avec `react-i18next` pour les applications React.

## Site Web / Lien
- [Site Officiel](https://www.i18next.com/)
- [Dépôt](https://github.com/i18next/i18next)
- [Extension React](https://react.i18next.com/)

## Installation
```bash
# Installer i18next et l'extension React
npm install i18next react-i18next
```

## Commandes Utiles / Astuces
- `t('key')`: Fonction standard pour traduire une clé.
- `i18n.changeLanguage('fr')`: Méthode pour changer la langue actuelle dynamiquement.

## Configuration
- Généralement initialisé dans un fichier dédié (par exemple, `i18n.js`) où vous configurez les ressources (traductions), la langue par défaut et les options d'interpolation.

## Notes
i18next va au-delà des fonctionnalités i18n standard ; il inclut un support pour la pluralisation, le contexte et l'interpolation.

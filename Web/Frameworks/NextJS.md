# Next.js

> [!TIP]
> [Passer à la version française](#version-française)

## Description
Next.js is an open-source React framework designed for building full-stack web applications. It extends React with additional features and optimizations, including server-side rendering (SSR), static site generation (SSG), and automatic configuration of lower-level tools like bundlers and compilers. This allows developers to create interactive, dynamic, and fast React applications, improving performance and search engine optimization (SEO).

## Website / Link
- [Official Website](https://nextjs.org/)
- [Repository](https://github.com/vercel/next.js)

## Installation
```bash
# Create a new Next.js app
npx create-next-app@latest
```

## Useful Commands / Tips
- `npm run dev`: Starts the development server.
- `npm run build`: Builds the application for production.
- `npm run start`: Starts the production server.
- `npm run lint`: Runs the linter configuration.

## Configuration
- `next.config.js` (or `next.config.mjs`): Main configuration file for Next.js options.

## Notes
Next.js is the go-to framework for React development today, offering a robust solution for routing, data fetching, and rendering strategies.

# Version Française

## Description
Next.js est un framework React open-source conçu pour créer des applications web full-stack. Il étend React avec des fonctionnalités et des optimisations supplémentaires, notamment le rendu côté serveur (SSR), la génération de sites statiques (SSG) et la configuration automatique d'outils de bas niveau comme les bundlers et les compilateurs. Cela permet aux développeurs de créer des applications React interactives, dynamiques et rapides, améliorant ainsi les performances et l'optimisation pour les moteurs de recherche (SEO).

## Site Web / Lien
- [Site Officiel](https://nextjs.org/)
- [Dépôt](https://github.com/vercel/next.js)

## Installation
```bash
# Créer une nouvelle application Next.js
npx create-next-app@latest
```

## Commandes Utiles / Astuces
- `npm run dev`: Démarre le serveur de développement.
- `npm run build`: Construit l'application pour la production.
- `npm run start`: Démarre le serveur de production.
- `npm run lint`: Exécute la configuration du linter.

## Configuration
- `next.config.js` (ou `next.config.mjs`): Fichier de configuration principal pour les options de Next.js.

## Notes
Next.js est aujourd'hui le framework incontournable pour le développement React, offrant une solution robuste pour le routage, la récupération de données et les stratégies de rendu.

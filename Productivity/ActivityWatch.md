# ActivityWatch

> [!TIP]
> [Passer à la version française](#version-française)

## Description
ActivityWatch is an open-source automated time-tracking software. It tracks how you spend time on your devices without compromising your privacy. Unlike many other time trackers, it keeps all your data locally on your device, ensuring that no sensitive information is sent to the cloud.

## Website / Link
- [Official Website](https://activitywatch.net/)
- [Repository](https://github.com/ActivityWatch/activitywatch)
- [Documentation](https://docs.activitywatch.net/)

## Installation
Download directly from the official website or GitHub releases page.
It is available for Windows, macOS, and Linux.
```bash
# Download the installer or zip file for your OS from the website.
```

## Useful Commands / Tips
- **Dashboard**: View your daily activity timeline, top applications, and top browser domains.
- **Categories**: Create rules to automatically categorize your activities (e.g., Work, Social Media, Dev, Games).
- **Timeline**: Visualize exactly when you were active and what you were doing.
- **Watchers**: Extend functionality with watchers for specific editors (VS Code, Vim) or media players.

## Configuration
- **Web Watcher**: Install the browser extension (ActivityWatch-watcher-web) for detailed tracking of specific URLs and titles within Chrome/Firefox.
- **Categorization**: Spend time initially setting up your regex rules for categories to make the data meaningful.

## Notes
- **Privacy First**: Completely open-source and local data storage.
- **No Cloud Sync**: By default, there is no cloud sync, which is a feature, not a bug, for privacy-conscious users.
- **Raw Data**: You have access to your raw data in JSON format if you want to perform your own analysis.

# Version Française

## Description
ActivityWatch est un logiciel de suivi du temps (time-tracking) automatisé et open-source. Il suit la façon dont vous passez du temps sur vos appareils sans compromettre votre vie privée. Contrairement à de nombreux autres trackers, il conserve toutes vos données localement sur votre machine, garantissant qu'aucune information sensible n'est envoyée dans le cloud.

## Site Web / Lien
- [Site Officiel](https://activitywatch.net/)
- [Dépôt](https://github.com/ActivityWatch/activitywatch)
- [Documentation](https://docs.activitywatch.net/)

## Installation
Téléchargez directement depuis le site officiel ou la page des releases GitHub.
Il est disponible pour Windows, macOS et Linux.
```bash
# Téléchargez l'installateur ou le fichier zip pour votre OS depuis le site web.
```

## Commandes Utiles / Astuces
- **Tableau de bord (Dashboard)** : Visualisez votre chronologie d'activité quotidienne, vos applications principales et vos domaines de navigation principaux.
- **Catégories** : Créez des règles pour classer automatiquement vos activités (ex : Travail, Réseaux Sociaux, Dév, Jeux).
- **Chronologie (Timeline)** : Visualisez exactement quand vous étiez actif et ce que vous faisiez.
- **Watchers** : Étendez les fonctionnalités avec des observateurs pour des éditeurs spécifiques (VS Code, Vim) ou des lecteurs multimédias.

## Configuration
- **Web Watcher** : Installez l'extension de navigateur (ActivityWatch-watcher-web) pour un suivi détaillé des URL et des titres spécifiques dans Chrome/Firefox.
- **Catégorisation** : Passez du temps au début à configurer vos règles regex pour les catégories afin de rendre les données significatives.

## Notes
- **Priorité à la vie privée** : Totalement open-source et stockage des données local.
- **Pas de Cloud Sync** : Par défaut, il n'y a pas de synchronisation cloud, ce qui est une fonctionnalité, pas un bug, pour les utilisateurs soucieux de leur confidentialité.
- **Données Brutes** : Vous avez accès à vos données brutes au format JSON si vous souhaitez effectuer votre propre analyse.
